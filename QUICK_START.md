## 🚀 HOW TO RUN

### Quick Start
```bash
# Navigate to project
cd /var/www/mekatronikaGAME

# Open in browser
# http://localhost:8000
```

```bash
# Using PHP
php -S localhost:8000

# Using Node.js
npx http-server -p 8000

# Using Live Server (VS Code extension)
# Right-click index.html → "Open with Live Server"
```
